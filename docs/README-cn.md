## 环境需求 (测试过的 / 开发平台)

-
    - `开发平台1 Windows 10`
    - `Python 3.8.10`
    - `Windows10 x64`
    - `pyglet 2.0.dev9`
    - `Json5 0.9.6`
    - `pillow 8.1.0`
    - `AMD R5 5600X(AMD YES!)`
    - `AMD RX 550 4G`
-
    - `开发平台2 macOS Big Sur`
    - `Python 3.8.7`
    - `macOS Big Sur 11.2.1`
    - `Pyglet 1.5.15`
    - `Json5 0.9.5`
    - `pillow 8.1.2`
    - `Intel I5 8279U`

## 需要的Python模块

- json5 (已经内置V0.9.6 路径：`./libs/json5`)
- pyglet (已经内置V2.0.dev9 路径：`./libs/pyglet`)
- pillow
- semver
